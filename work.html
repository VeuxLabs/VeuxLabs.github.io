  • Lightspeed FlightLink System

    Project

    FlightLink Connected System
    iOS Universal App

    Client

    Lightspeed Navigation

    Challenge

    Streamlining In-Flight Communication
    Flying an airplane is no easy matter. Pilots must constantly monitor dozens of gauges and indicators in the cockpit while receiving updates from air traffic control. A misheard communication adds stress and risk to the flying experience. What could improve the flow of information between tower and pilot?

    Main View with Writing Pad

    Solution

    FLYING WITH APP SUPPORT
    Veux Labs began by interviewing pilots and observing them on several flights. We realized that an app offering a simple playback would answer most plane-to-tower concerns: if pilots could access recent communication with the touch of a button, they wouldn’t need to ask air control to repeat updates. We quickly identified the iPad as the ideal support device, since iPads are commonly used for pre-flight checks and in-flight navigation.

    Headset Connection and Setup

    Collecting Communications
    A unique hardware feature allows all cockpit intercom and radio traffic to be mixed and then output to the iPad's auxiliary port or Bluetooth connection, where it is then recorded by the FlightLink app. Certified under Apple’s Made For iPod (MFi) program, FlightLink is a universal app for iPad, iPhone, and iPod Touch.

    Recording Library
    Night Mode
    Playback Mode
    Playback Mode
    Recording Library
    Night Mode

    One-Touch Access
    Given the complex environment of the cockpit, FlightLink had to be easy to use. The user interface allows simple one-touch access to any segment of the last two minutes of recorded audio. A digital notepad enables the pilot to jot down instructions or other relevant information.

  • Mitel TCO Web App

    Project

    Total Cost of Ownership Web App
    Responsive Web App for Mobile
    and Desktop

    Client

    Mitel & Unified Communications Strategies

    Challenge

    ENABLING THE FIELD SALES TEAM
    The Unified Communications marketplace is crowded, and competitors all seem to be saying the same thing, making it hard for customers to choose a partner for their UC requirements. Mitel has a distinct advantage in Total Cost of Ownership, but needed a way to enable their field sales team and channel partners to easily communicate that advantage to their potential customers.

    Solution

    Login
    Data Entry
    System Analysis Result
    TCO Analysis Results

    CREATIVE COLLABORATION
    Mitel worked with UC Strategies, a strategic consulting firm specializing in enterprise unified communications, to create a total cost model that accurately predicts customer savings versus Cicso, Avaya and Microsoft, who are Mitel's primary competition. Then Veux Labs and UC Strategies worked together to create a mobile-first, responsive web application that can be used on any mobile or desktop platform.

    EASY YET SOPHISTICATED
    Veux Labs created a simple–to–use interface sitting on top of a very sophisticated cost model. The interface made it possible for a sales partner to enter six values and generate detailed comparison graphs that can be easily shared with the customer.

    iPad Login
    iPad Data Entry

    MOBILE FIRST
    We used a mobile-first approach to design to ensure the web app is optimized for the small screen, using progressive enhancement to modify the screen configuration on the tablet and desktop sized screens where more area is available for navigation and information.
